内部存货交易未实现利润自动抵消系统——演示指南
====================================================
2025级会计专硕 课程作业

【环境准备（首次使用，仅需做一次）】

1. 安装 Python
   - 打开浏览器，访问：https://www.python.org/downloads/
   - 点击黄色按钮 "Download Python 3.x.x" 下载
   - 双击下载的文件安装
   - ★ 关键：安装界面底部勾选 ✅ "Add Python to PATH"
   - 点击 Install Now，等待完成

2. 验证安装
   - 按键盘 Win+R，输入 cmd，回车
   - 在黑窗口输入：python --version
   - 如果显示 "Python 3.x.x" 说明安装成功


【启动系统】

1. 双击文件夹里的 "启动系统.bat"
   - 首次运行会自动安装依赖（约1-2分钟，等待黑窗口里的进度条跑完）
   - 之后运行秒开

2. 浏览器自动打开 http://localhost:8501
   - 如果没有自动打开，手动在浏览器输入这个地址

3. 使用系统
   方式A（推荐）：点击左侧 "一键加载示例数据" → 点 "执行抵消计算"
   方式B：手动上传三份Excel（templates 文件夹里有模板）
   方式C：在页面"手动录入交易"区域直接填表添加


【演示重点】

1. ★ 抵消分录自动生成 —— 系统核心功能
   上传数据后自动输出标准借贷分录，含递延所得税 + 少数股东分摊

2. 仪表盘
   顶部5个KPI卡片 + 两张柱状图 + 抵消前后对比表

3. 异常检测
   如果数据有问题（负毛利、库存缺失等），系统自动预警

4. 导出
   支持导出 Excel 和 PDF 报告


【停止系统】

关闭黑窗口即可。


【常见问题】

Q: 双击bat闪退？
A: 可能Python没装好。按 Win+R → 输入 cmd → 输入 python --version 检查

Q: 浏览器显示 "无法访问"？
A: 等黑窗口出现 "You can now view your Streamlit app" 再刷新浏览器

Q: 想用自己的数据？
A: 下载 templates 文件夹里的Excel模板，按格式填数据后上传
